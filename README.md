# Django AI Chatbot (Groq API)

A simple AI chatbot built using **Django (Function Based Views)** and the **Groq AI API**.  
This project demonstrates how to integrate an AI API into a Django backend using clean server-side rendering (no JavaScript).

## Features
- Django Function Based Views
- AI responses powered by Groq (LLaMA models)
- No JavaScript used
- Simple and clean chat interface
- Secure API key handling with `.env`

## Tech Stack
- Python
- Django
- Groq API
- HTML & CSS
- SQLite (development)
